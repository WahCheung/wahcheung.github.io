贝叶斯公式

发表于 2017-11-25 | 分类于 机器学习

全概率定理

设$A_1,A_2,\cdots,A_n$是一组互不相容的事件，形成样本空间的一个分割（每一个实验结果必定使得其中一个事件发生）。又假定对每一个$i$，$P(A_i) > 0$。则对于任何事件$B$，下列公式成立
$$P(B) = P(A_1 \cap B) + \cdots + P(A_n \cap B) = P(A_1)P(B \mid A_1) + \cdots + P(A_n)P(B \mid A_n)$$
